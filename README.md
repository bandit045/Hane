## Download

To clone whole repository with commit, and diff-s use: `git clone --recursive https://github.com/bandit045/Hane.git`

To clone latest update only, and save memory use: `git clone --recursive --depth 1 https://github.com/bandit045/Hane.git`

## Premake5

Project (solution) can be build for various IDE.
Mainly for VS2019
But can be easily switched by just change inside `Start.bat` which is supported by Premake5
```.bat
@echo off
pushd premake
call premake5.exe vs2019
PAUSE
```
by changing vs2019 to:
- `vs2017`
- `vs2019` - active
- `vs2022`
- `vs2026`
